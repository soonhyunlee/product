# ⚡ AI Product Name Refactorer (상품명 초간결 정제 솔루션)

> **상세페이지 크롤링 분석 및 상품명 7자 내외 초간결 최적화 정제 소프트웨어**입니다.  
> 윈도우 환경 및 GUI 라이브러리 제약을 극복하고, Flask와 HTML5/CSS3/JS 기반의 **최첨단 웹 브라우저 대화형 화면(Web GUI)**을 제공합니다.

---

## ✨ 핵심 기능 (Features)

1. **지능형 브랜드 및 광고어 일괄 정제**
   * 미리 확장 등록된 제조사/브랜드 필터(`3M`, `오공`, `일신` 등)와 다양한 광고 수식어(`품절`, `특가`, `최저가` 등)를 원천 차단하여 순수한 핵심 상품 단어만 추출합니다.
2. **7자 내외 초정밀 압축 알고리즘 (`refactor_to_concise_name`)**
   * 상품명이 6글자를 초과할 때, **'핵심 상품 명사 + 필수 고유 모델코드 + 규격 단위'** 구조만 지능적으로 보존하고 부사어 및 불필요한 단어를 과감히 삭제하여 글자 수를 단축시킵니다.
   * `개입`, `매입` 등을 `개`, `매`로 실용 변환하고 다중 공백을 완벽 제거합니다.
3. **모델 코드 기반 70~80% 유사성 우회 판별**
   * A열의 원본명이 단순 규격 코드(예: `LP-7000`)처럼 부실하더라도, 상세페이지 크롤링 결과와 **핵심 모델 코드**가 교차 일치할 경우 **70~80%의 유사 연관성**을 자동 확인하여 신뢰성 필터를 우회 통과시킵니다.
4. **프리미엄 로컬 웹 GUI (Web GUI Interface)**
   * `tkinter` 창 그리기 라이브러리 없이도 구동이 가능한 웹 브라우저 환경입니다.
   * 마우스 더블클릭 한 번으로 기본 크롬/엣지 브라우저가 자동 연동 실행됩니다.
   * **실시간 진행 게이지** 및 **에메랄드 터미널 모니터링 로그**를 통해 스캔 과정을 한눈에 모니터링할 수 있습니다.

## 🛠️ 설치 및 사용법 (How to Run)

### 1. 프로그램 즉시 구동 (가장 간편한 방법)
프로젝트 루트 디렉토리에 포함된 **`실행하기.bat`** 파일을 파일 탐색기에서 **더블클릭**합니다.
* 자동으로 로컬 웹 서버가 가동되고, 브라우저가 열리며 아래의 웹 페이지 주소가 로드됩니다.
* 주소: `http://127.0.0.1:5000`

### 2. 터미널 명령어로 직접 실행
윈도우 환경에서 특정 파이썬 바이너리를 지정하여 직접 실행할 수 있습니다.
```powershell
"C:\Program Files\PostgreSQL\18\pgAdmin 4\python\python.exe" c:\product\excel_cleaner.py
```

## 📂 파일 구조 (File Structure)

* `excel_cleaner.py`: 크롤링, 7자 정제 압축 코어 로직 및 Flask 로컬 웹 서버 코드가 담긴 메인 스크립트.
* `실행하기.bat`: 원클릭 구동이 가능한 Windows 배치 실행기 (ANSI CP949 인코딩으로 안전 설계).
* `create_sample.py`: 35종의 고난도 테스트용 샘플 엑셀 파일을 자동 빌드해 주는 샘플 생성 스크립트.
* `.gitignore`: 불필요한 캐시 파일 및 정제 결과 엑셀 파일이 Git에 업로드되는 것을 차단하는 무결성 설정 파일.
